===================

Welcome to the Issue #99 of *Swift for TensorFlow Newsletter*. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

## S4TF Commits

#### Swift Compiler (main branch)

* [PR-35594](https://github.com/apple/swift/pull/35594): Remove a bit of dead code in `DerivedConformanceDifferentiable.cpp`.

#### Swift TensorFlow APIs

* [PR-1129](http://github.com/tensorflow/swift-apis/pull/1129): Adds `dilations` argument inside `DepthwiseConv2D` API.

#### Swift Models

* [PR-631](https://github.com/tensorflow/swift-models/pull/631): Introduces a higher-level interface to model checkpointing.

* [PR-731](https://github.com/tensorflow/swift-models/pull/731): Uses respectful language in code base.
