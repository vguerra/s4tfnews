S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-28150: Extends test suite to cover Crashers, Activity analysis and Control-Flow differentiation.

  • PR-28067: IRGen differentiability witness tables. Resolves TF-930.

  • PR-28117: Introduces usage of PrettyStackTrace in Differentiation transform.

  • PR-28089: Moves differentiation sources in stdlib into stdlib/public/core/Differentiation.

  • PR-28069: Removes explicit flag -enable-large-loadable-types from tests.

  • PR-28018: Fixes building the macOS toolchain build-toolchain-tensorflow.

Swift TensorFlow APIs

  • PR-288: Fixes bug in TransposedConv2D and adds respective tests.

  • PR-525: Changes type of some properties from Tensor<Scalar> to Scalar.

  • PR-541: Moves Dense and Dropout Layers into separate files.

  • PR-523: Adds preconditions for conv2D. Partially resolves Issue-517.

  • PR-532: Adds support for constant and truncated Normal initializers.

  • PR-540: Introduces the Tensor library.

  • PR-547, PR-544, PR-543: Renames enum Raw to _Raw to make explicit that it is a unstable / not-guaranteed API.

Swift Models

  • PR-220: Follows up on changes introduced by tensorflow/swift-apis#525.
