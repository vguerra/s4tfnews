S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-29707: Fixes Jupyter completion API.

  • PR-29689: Adds support for non-active switch_enum_addr differentiation. Resolves TF-1132.

  • PR-29663: Fixes REPL crash involving pattern bindings to expressions with closures. Resolves TF-1144.

  • PR-29571: Removes quasinotes implementation from tensorflow branch.

  • PR-29647: Fixes @noDerivative attributes in non-primary-files, all @noDerivative attributes are lowered to [_semantics "autodiff.nonvarying"] SIL function attributes.

  • PR-29621: Improves getConstrainedDerivativeGenericSignature naming and documentation.

Swift TensorFlow APIs

  • PR-599: Adds triangularSolve operation and its gradient.

  • PR-663: Add initialState wrt parameter for RNN.

  • PR-579: Improves documentation for losses by adding equations.

  • PR-646: Introduces activation function swish.

  • PR-660: Fixes derivative of RNN.callAsFunction(_:initialState:).

  • PR-658: Makes sure that swift files are installed into the OS specific path.

  • PR-655: Adds support for PythonKit as an alternative to Python.

Swift Jupyter

  • PR-86: Allows displaying of messages with custom mimetype data.

  • PR-85: Adds convenience functions for drawing SwiftPlot plots.

Swift Models
