S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-28930: Replaces SIMD use of @differentiable(jvp:vjp:) with @derivative(of:).

  • PR-28935: Fixes @derivative(of: subscript) crash for SIMD{n} types. Resolves TF-1090.

Swift TensorFlow APIs

  • PR-601: Adds precondition to matmul to check that Tensor has at least rank 2. Partially resolves Issue-517.

  • PR-602: Fixes a bug in VJP of transposed(permutation:).

Swift Models

  • PR-270: Enables user inputting source url while creating CIFAR10.

  • PR-261: Introduces DCGAN.
