S4TF News - Issue 30, October 6th 2019

Welcome to the Issue #30 of Swift for TensorFlow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

News and Articles

S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-27556: Adds tests for AutoDiffIndexSubset initializers.

  • PR-27514: Replaces AutoDiffParameterIndices with AutoDiffIndexSubset and deletes AutoDiffParameterIndices. Resolves TF-538

  • PR-27505: Removes Differentiable APIs deprecated in release of v0.5.

  • PR-27472: Makes sure that activity analysis does not propagate activity through @noDerivative field projections from a struct. Resolves TF-865.

  • PR-27471: Renames AutoDiffFunction to DifferentiableFunction.

  • PR-27428: Optimizes computation of derivative generic signatures.

  • PR-27445: Adds -enable-experimental-differentiable-programming frontend flag to enable all differentiable programming features in the compiler.

Swift TensorFlow APIs

  • PR-520: Deprecates Tensor.droppingOut.

  • PR-514: Introduces Tensor padding modes.

Swift Models

  • PR-213: Introduces DenseNet(DenseNet121) implementation.

Swift for Tensorflow Docs

  • PR-233: Adds script that install a Swift for TensorFlow toolchain on Ubuntu 18.04.
