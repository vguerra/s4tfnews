S4TF News - Issue 87, November 8th 2020

Welcome to the Issue #87 of Swift for TensorFlow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

News and Articles

  • The 1st release candidate for Swift for TensorFlow 0.12.0 is now available. Grab a copy from the Release Candidate's section from the installation instructions. The S4TF v0.12 Release Notes document guides you through all changes since v0.11, new changes and known issues. Please give this release a try and report any issues you find on JIRA or GitHub issues on Swift-APIs.

  • On Friday 6th of November, the 43rd SIG Swift Design meeting took place. Here you have the highlights:

    • Denys Shabalin gave an introductory talk about Swift Benchmark: A Swift library for benchmarking code snippets. The slides of the presentation are available online.

    You can find the notes for this meeting in this google doc.

    Recording from this session is available online.
