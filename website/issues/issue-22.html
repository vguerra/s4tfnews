    • Eugene Burmako presented the initial work on Swift as Syntactic sugar for MLIR which aims at explore ways to metaprogram MLIR in Swift. Check out as well his post on the Swift forums.

    • Anthony Platanios gave an overview on his proposal PR-431 which adds sipport for learning rate schedules.

    • If you missed the previous meeting, you can catch up watching the recording, specially if you are interested in the AD implementation overview.

    You can find the meeting notes and all questions raised in this google doc.

S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-26595: Adds mangling support for @differentiable function tyopes. Allows as well to build macOS toolchains with --release-debuginfo flag. Resolves PR-TF-597.

  • PR-26553: Fixes wrong ordering on pullback results. Resolves TF-742.

  • PR-26518: Introduces experimental prototype of Swift quasiquotes.

  • PR-26527: Deprecates AllDifferentiableVariables associated type of Differentiable protocol. Related PR: PR-419. Resolves (TF-707).

  • PR-26484: Fixes memory leak when noescape closure being differentiated captures a mutable self.

  • PR-26503: Enables ownership for AD linear map SILGen thunks.

  • PR-26448: Fixes self re-ordering bug of SILGen JVP/VJP thunks . Resolves TF-698.

  • PR-26447: Removes deprecated tests in TensorFlowUnittest and AD control flow tests as they have been moved to the [swift-api](https://github.com/tensorflow/swift-apis)'s repo (PR-408) .

  • c9672311: Improves reabstraction of a @differentiable function during SIL generation by avoiding copies of the original @differentiable function.

Swift TensorFlow APIs

  • PR-438: Introduces test for BatchNorm inference.

  • PR-428: Fixes bugs related to shapes in nomalization layers. Resolves PR-384, PR-426.

  • Added activation functions:

    • PR-435: relu6.

    • PR-430: selu.

  • PR-433: Substitutes usage of Raw.add by Raw.addV2. The later is makred is_commutative & is_aggregate which might be used for optimizations by Grappler (Default graph optimization system in the TF runtime).

  • PR-362: Introduces SeparableConv2D Layer.

  • PR-351: Adds support for negavite axis argument values in Tensor.unstacked(alongAxis:).

  • PR-419: Removes usage of AllDifferentiableVariables and var allDifferentiableVariables. Related PRs: PR-420, PR-26527.

  • PR-415: Adds tests for LSTMCell.

  • PR-388: Adds shape tracking in LazyTensor.

  • PR-414: Fixes bug on initialization of bias in DenseLayer.

Swift Models

  • PR-189: Adds implementation of SqueezeNet.

Swift for Tensorflow Docs

  • PR-187: Adds a tutorial on Protocol oriented programming & Generics in Swift.

Swift Jupyter

  • PR-70: Fixes .display method lookup issue.
