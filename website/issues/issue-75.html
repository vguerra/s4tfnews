S4TF News - Issue 75, August 16th 2020

News and Articles

Please give this release a try and report any issues you find on GitHub issues for Swift-APIs repository.

S4TF Commits

Swift Compiler (master branch)

  • PR-33483: Adds differentiation support for non-active try_apply SIL instructions. Resolves TF-433.

  • PR-33386: Fix Optional differentiation crash for non-resilient Wrapped reference type. Resolves SR-13377.

Swift TensorFlow APIs

  • PR-1060: Disables swift-numerics on Darwin targets.

  • PR-1055: Define Tensor.zeroTangentVectorInitializer that returns a zero tensor with the same shape as self, instead of a scalar zero tensor.

Swift Models

  • PR-654: Adds 3 new arguments for specifying platform for benchmarks: --cpu, --gpu, --tpu.

  • PR-617: Introduces implementation of Deep Q-Network.
