S4TF News - Issue 67, June 21st 2020

Welcome to the Issue #67 of Swift for TensorFlow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

News and Articles

  • The 1st release candidate for Swift for TensorFlow 0.10.0 is now available. Grab a copy from the Release Candidate's section from the installation instructions. The S4TF v0.10 Release Notes document guides you through all changes since v0.9, new changes and known issues. Please give this release a try and report any issues you find on JIRA or GitHub issues on Swift-APIs.

  • A new Xcode 11 Development snapshot is now available for you to try out Swift for Tensorflow. Get it here.
