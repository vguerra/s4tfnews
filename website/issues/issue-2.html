S4TF Commits

Swift Compiler (tensorflow branch)

  • Work related to @differentiating attribute:
    • PR-23384: Allows usage of @differentiating in a more generic context.
    • PR-23373: @differentiating cab be used on subscripts.
    • PR-23370: Alows @differentiating attribute to support wrt clauses.
    • Several bug fixes: PR-23379, PR-23383
    • PR-23193: Increasing usage of @differentiating within the standard library.
  • PR-23339: Allows TensorHandle Scalar to used from @inlinable code w/out exposure.
  • PR-23439: Marks flattened() and rankLifted() as differentiable.
  • PR-23355: Adds derivatives for reshaping methods.
  • PR-23268: Makes PythonObject conform to Sequence.
  • PR-23351: Allows convertion from Optionals to Python objects.

Swift TensorFlow APIs

  • PR-58: Introduces Backpropagator typealias.

  • PR-57: Introduces 3 news layers: Conv1D, MaxPool1D and AvgPool1D.
