S4TF News - Issue 0, March 10th 2019

Welcome to the Issue #0 of Swift for Tensorflow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for Tensorflow project.

Enjoy!

News and Articles

  • Version 0.2 of the pre-built packages was released on March 1st.

  • Check out the TensorFlow Dev Summit '19 presentation from Chris Lattner and Brennan Saeta about Swift for Tensorflow: The Next-Generation Machine Learning Framework. You can as well have a look at the notebooks used during the presentation in github.

  • During the TensorFlow Dev Summit '19 [fast.ai]|(https://www.fast.ai/) announced that they will embrace Swift for Tensorflow in a couple of their lessons. Even more exciting: Chris Lattner will be co-teaching them. Check out the official announcement written by Jeremy Howard.

  • Tanmay Bakshi has been sharing on his youtube channel great videos on how to use Swift for Tensorflow. Check them out! So far he's got: Python interoperability, a CNN for digit classification and OpenAI-Unsupervised Sentiment Neuron.

  • Jacopo Mangiavacchi has shared on Twitter a bunch of sample notebooks showcasing Swift for Tensorflow, check them out.

  • Brad Larson writes about his experience on training AlexNet using Swift for TensorFlow. You can find his code on github.

The mailing list

  • This year the Tensorflow project has been accepted as an official open source mentor and among the proposed ideas to work on you can find 4 projects under the Swift for Tensorflow umbrella. Read all about it in the official announcement in the mailing list.

  • Richard Wei shared the new project page, API documentation and several tutorials that were released during the Tensorflow Dev Summit '19. (original post).

  • Volodymyr Pavliukevych shared his swift package to track and visualize Tensors in Tensorboard. He is calling his package: KrakenKit. (original post).
