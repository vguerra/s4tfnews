S4TF Commits

Swift Compiler (master branch)

  • PR-32110: Enable .swiftinterface verification for _Differentiation.

  • PR-32106: Upstream of forward-mode differentiation runtime tests.

  • PR-32031: Fixes differentiation crashes related to definite initialization. Resolves SR-12886 and SR-12887.

  • PR-31823: Adds support to derive Differentiable.zeroTangentVectorInitializer. Resolves TF-1007.

  • PR-32069: Adds support differentiation of branching cast instructions ( operators is and as? operators ). Resolves SR-12898.

  • PR-32003: Fixes false Differentiable synthesis warning on class property wrappers. Resolves SR-12873.

Swift Compiler (tensorflow branch)

  • PR-32076: Adds StringSwitch to LLVMSupport fork.

  • PR-32066: Isolates further runtime from LLVMSupport.

Swift TensorFlow APIs

  • PR-957: Hides the symbols except for those that are marked as visible. Partially resolves TF-1273

  • PR-976: Removes reliance on experimental TF_GetServerDef TensorFlow api. Partially resolves Issue-970.

  • PR-969: Adds tests for Slices. Resolves Issue-968.

  • PR-967: Fixes bugs, corrects comments in Sampling.swift and adds tests.

Swift Models

  • PR-571: Makes sure to return replacement character instead of throwing error if bpe decode fails. Resolves Issue-385.

  • PR-573: Throws error if GPT2 checkpoint fails to load. Resolves Issue-568.

  • PR-572: Removes executable products but leaves them as targets. Resolves Issue-561.

  • PR-564: Introduces Rectified Adam optimizer.
