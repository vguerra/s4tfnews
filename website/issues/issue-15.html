S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-25688: Enforces @differentiable closure converstion to be applied only when from-expression is a literal closure expression or a reference to a func declaration.

  • PR-25594: Allows you to reference parameters by index w.r.t order of appearence in original function in wrt: argument of attributes @differentiating and @differentiable.

  • PR-25558: Introduces differentiation for loops.

  • PR-25552: Adds top-level withoutDerivative(at:in:) to support generic algorithms that are conditionally differentiable.

  • PR-25525: Adds broadcasting methods adding(_:) and substracting(_:) to protocol VectorProtocol which are compiler-derivable.

  • PR-25509: Adds support for diferentiation of switch_enum terminator. Differentiation of active enum values is not yet supported though.

Swift TensorFlow APIs

  • PR-150: Introduces logSumExp which is more numerically stable than log(exp(self).sum(...)).

  • PR-278: Improves error message for when computing gradient of a function does not return a scalar.

  • PR-275: Adds support for dilated Conv1D and Conv2D.

  • PR-272: Makes 'Tensor.gathering' generic over Int32 and Int64.

  • PR-157: Adds support for 'Tensor.batchGathering(atIndices:)'.

  • Added activation functions:

    • PR-268: Adds Gaussian Error Linear Unit (GELU) activation function gelu(_:).

    • PR-260: Adds leaky ReLU activation function leakyRelu(_:alpha:).

    • PR-252: Adds Exponential Linear Unit activation function elu(_:).

  • PR-264: Introduces contribution guidelines.

  • PR-257: Adds Embedding layer.

  • PR-202: Extract traces for evaluating a LazyTensorOperation.

  • PR-197: Introduces Glorot normal initialization for Tensor.

  • PR-251: Adds implementations for adding(_:) and subtracting(_:) of VectorProtocol protocol.

  • PR-241: Adds 2D depthwise convolution layer DepthwiseConv2D.

  • PR-227: Adds Logarithm-cosine loss logCoshLoss(predicted:,expected:).

Swift Jupyter

PR-66: Introduces %system directive to install libraries before swift packages.
