S4TF News - Issue 26, September 8th 2019

Welcome to the Issue #26 of Swift for TensorFlow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

News and Articles

  • The 1st release candidate for Swift for TensorFlow 0.5.0 is now available (Ubuntu and Colab only, macOS package is coming soon!). Grab a copy from the Release Candidate's section from the installation instructions. The S4TF v0.5 Migration Guide document guides you through all changes since v0.4. Please give this release a try and report any issues on JIRA you find.

  • On Monday 9th of September at 6 PM (Pacific Time), the Swift for TensorFlow meetup will host an event. The topic for this edition is Swift as syntactic sugar for MLIR. The event is taking place at Google Developers Launchpad Space in San Francisco, CA. If you plan to attend, make sure you reserve your place.

  • Richard Wei has posted in the Swift Evolution forums the proposal for Differentiable Programming in Swift.

  • James Dellinger has written an excellent guide on Building and Training an XResNet from Scratch.

  • The 2019 edition of GSoC has endeed and results have been announced this last 3rd September. Go to the TensorFlow organization page to access the final submitions the students worked on.
