S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-28921: Adds diagnostics for unsupported coroutine differentiation. Resolves TF-1081.

  • PR-28908: Makes public memberwise initializers of synthesized TangentVector structs to improving usability. Resolves TF-1077.

  • PR-28891: Introduces flag enable-experimental-cross-file-derivative-registration to enable cross-file derivative registration.

  • PR-28887: Renames @nondiff to @noDerivative.

  • PR-28889: Fixes bugs on Array literal differentiation. Resolves TF-976, TF-975 and TF-975.

  • PR-28785: Adds some @derivative(of:) tests.

  • PR-28791: Cleans up @transpose attribute printing and parsing.

Swift TensorFlow APIs

  • PR-591: Makes use of @derivative for derivative registration. Resolves TF-1076.

  • PR-590: Fixes bugs in Tensor.min(alongAxes:) and Tensor.min(squeezingAxes:).

  • PR-586: Adds trace linear operator. Partially solves Issue-562.

  • PR-557: Adds more preconditions for Basic.swift module. Partially solves Issue-517.

  • PR-280: Adds dilation factor to Conv3D.

  • PR-583: Adds categorical intializer for Tensor.

Swift Models

  • PRR-241: Adds the Imagenette dataset.

  • PR-251: Introduces VGG19 and rewrites VGG to use blocks.

  • PR-260: Changes datasets to use a temporary directory for downloads by default.

  • PR-257: Adds support for additional benchmark metrics: exp_per_second and avg_exp_per_second.
