S4TF News - Issue 50, February 23rd 2020

Welcome to the Issue #50 of Swift for TensorFlow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

News and Articles

  • The final release for Swift for TensorFlow 0.7.0 is now available. Grab a copy from the Release Candidate's section from the installation instructions. The S4TF v0.7 Release Notes document guides you through all changes since v0.6.

  • A new Xcode 11 Development snapshot is now available for you to try out Swift for Tensorflow. Get it here.

  • Once again, TensorFlow has been chosen as a mentoring organization for Google Summer of Code edition 2020. Here you have the list of propossed ideas for Swift for TensorFlow ( as well, you are encouraged to propose your own ideas ). If you would like to apply, have a look at this year's edition timeline.

S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-30015: Fixes --enable-tensorflow flag propagation from build-script to build-script-impl. Resolves TF-1174.

  • PR-30008: Removes deprecated @differentiating attribute. Resolves TF-999.

  • PR-29947: Makes build preset tensorflow_osx_base directly use buildbot_osx_package as a mixin . Resolves TF-941.

  • PR-29394: Adds a new abstraction pattern that properly abstracts the derivative functions in a differentiable_function bundle that is being abstracted to AP::Opaque. Resolves TF-123.

  • PR-29920: Handles begin_borrow in reapplyFunctionConversion. Resolves TF-1159.

  • PR-29908: Propagates @frozen and @usableFromInline to TangentVector struct.

Swift TensorFlow APIs

  • PR-692: Removes deprecated method-style differential operators.

  • PR-689: Introduces tiled operation.

  • PRr-688: Improves compilation times for tests.

  • PR-687: Adds preconditions for pooling layers and pooling functions. Partially resolves Issue-517.

  • PR-686: Introduces parameter embeddingsInitializer for Embedding layer as well as preconditions vocabularySize and embeddingSize. Partially resolves Issue-517.

  • PR-677: Adds precondition in BatchNorm and LayerNorm for offset and scale. Partially resolves Issue-517.

  • PR-674: Makes bias optional in Dense and all Conv layers.

Swift Models

  • PR-336: Splits the Transformer executable into a Transformer library and a TransformerDemo executable.

  • PR-335: Updates usage of deprecated method-style differential operators.

  • PR-316: Introduces Batcher.

  • PR-306: Adds documentation on CMake support.

  • PR-311: Adds Tabular Regression Model using Boston Housing dataset.

  • PR-302: Introduces FashionMNIST dataset.

  • PR-324: Adds Transfomer model to CMake based build.
