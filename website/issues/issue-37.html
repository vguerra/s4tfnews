S4TF News - Issue 37, November 24th 2019

Welcome to the Issue #37 of Swift for TensorFlow Newsletter. A newsletter letter that summarizes weekly activity around the Swift for TensorFlow project.

Enjoy!

News and Articles

  • Last September Brett Koonce gave a talk at Kirkland ML Summit 2019 titled Convolutional Neural Networks with Swift. The recording of the talk is available on YouTube. Among the examples he shows on this talk, you can find swift MNIST, which is a Swift Package for the MNIST Dataset.

  • The 1st Release Candidate for Swift for TensorFlow toolchain is available for download. Grab your copy from the Releases section from the Install instructions. The release notes are available in this document. Feel free to report any issues on the mailing list or create tickets on the respective tracking systems: Swift Compiler, DeepLearning Library.

S4TF Commits

Swift Compiler (tensorflow branch)

  • PR-28429: Improves diagnostic for multiple active results.

  • PR-28402: Populates differentiability witnesses during the differentiation pass.

  • PR-28389: Introduces tests for differentiability witness wrt params.

  • PR-28108: Introduces derivatives for more tgmath functions. Resolves TF-812.

  • PR-28352: Fixes variedness propagation for apply inout arguments. Resolves TF-974.

  • PR-28320: Revamps Array literal initialization differentiation. Resolves TF-952.

  • PR-28331: Implements IRGen for differentiability_witness_function.

Swift TensorFlow APIs

  • PR-563: Introduces support for cholesky decomposition, including it's derivative.

  • PR-250: Adds @_Freezable property wrapper.

  • PR-174: Introduces layers TransposedConv1D and TransposedConv3D.

  • PR-335: Refactors random and variance-scaling initializers.

  • PR-558: Moves TensorHandle related code from ShapedArray.swift to TensorHandle.swift.

Swift Models

  • PR-226: Add support for CL arguemnt parsing on Benchmarks target.

  • PR-224: Introduces ImageClassificationDataset protocol and makes MNIST and CIFAR10 Datasets conform to it. Resolves Issue-206.

Swift Jupyter

  • PR-82: Fixes build.db lookup.
